# Design Inspiration Skill

When an agent or user needs **UI/UX or design inspiration** for the target product or similar products, apply this skill. It surfaces curated "start here" lists for four focus areas: (1) Infographics, (2) Landing pages, (3) Design systems, (4) Templates for building infographics. Reference the relevant section and tables below; keep URLs and one-line descriptions exact when citing resources.

---

## When to use this skill

- User or agent asks for **design inspiration**, **UI/UX inspiration**, or **visual reference** for infographics, landing pages, or builder UIs.
- Working on the target product or an **infographic generator** and need layout ideas, chart-choice guidance, or best practices.
- Need **landing page** inspiration for SaaS, creator tools, or design products.
- Building or refining UI (forms, controls, charts, layout) and need **design system** or **component library** references.
- Need **templates** (Figma, Canva, etc.) or UI kits for charts, one-pagers, reports, or social infographics.
