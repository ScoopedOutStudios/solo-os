# Weekly Solo OS Review Checklist

- Run `solo-os sync-audit`
- Review `reports/latest-sync-audit.json`
- Ensure every active initiative has:
  - canonical plan pointer
  - execution links section
  - at least one validation artifact
- Run `solo-os cleanup-markdown` (dry-run)
- Apply markdown cleanup if candidates are confirmed
- Park or archive low-leverage backlog items
- Confirm top 1-3 tasks for next workday
