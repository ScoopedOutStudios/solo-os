# Agent-Generated Artifacts

This directory contains non-canonical drafts, generated analysis, and archive material for this repo.

## Folder Structure

| Folder | Purpose |
|---|---|
| `drafts/` | Temporary working drafts and intermediate generated notes |
| `archive/` | Superseded or low-value historical material |
| `plans/` | Pointer-only execution-plan compatibility surface |

## Rules

- Follow `sos-hq/docs/governance/artifact-governance-spec.md` for metadata, versioning, and lifecycle.
- Keep canonical roadmap, product, and decision narrative in `docs/`, not here.
- Keep `_index.md` registry-only.
- Keep `plans/LATEST.md` pointer-only.
