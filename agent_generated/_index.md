# Agent Artifacts Index

Registry-only pointer surface for generated and compatibility artifacts in this repo.

Policy source:
- `sos-hq/docs/governance/artifact-governance-spec.md`
- `agent_generated/README.md`

## Current Registry

| Surface | Current Pointer | Purpose |
|---|---|---|
| roadmap and strategy | `docs/product/` | Canonical roadmap, positioning, and long-form product narrative |
| research | `docs/research/` | Durable discovery, validation, and analysis synthesis |
| decisions | `docs/decisions/` | Canonical major decisions and decision history |
| economics | `docs/economics/` | Pricing and unit economics reasoning |
| active execution plan | `agent_generated/plans/LATEST.md` | Pointer to the current build loop when one exists |
| generated archive | `agent_generated/archive/` | Superseded and low-value historical material |

## Update Rules

- Keep this file registry-only. Do not add status boards, narrative summaries, or backlog management here.
- Put active workflow state in GitHub Projects/Issues.
- Update this file only when canonical paths materially change.
